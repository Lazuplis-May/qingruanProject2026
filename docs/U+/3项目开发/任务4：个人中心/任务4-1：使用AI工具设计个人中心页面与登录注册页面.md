任务描述

**1. 任务描述**

在个人中心中可以进行个人信息编辑、查看生活方案、我的资讯、打卡记录、帮助中心，如果登录用户为管理员，则还可以通过个人中心进入AI智能管理平台。进入个人中心前，还需要完成注册和登录操作。

**难点 ：**使用AI工具完成个人中心与注册登录界面设计；

任务指导

#### 1.个人中心页面设计

通过AI工具设计个人中心页面，该页面包括用户头像、用户名、操作列表、退出按钮。提示词参考如下：

```
生成一个个人中心页面，页面包括头像，个人名称，菜单选项栏，退出按钮，菜单选项栏包括个人信息、我的方案、我的资讯、打卡记录、帮助中心、AI智能管理平台。界面简介美观，以蓝白为主色调。
```

#### 2.登录注册页面设计

在登录注册界面中，点击登录或注册按钮切换不同的表单，登录页面输入内容包含用户名和密码，注册页面输入内容包含用户名、确认密码和密码。可参考以下提示词。

```
生成登录注册页面。登录注册界面中，点击登录或注册按钮切换不同的表单，登录页面输入内容包含用户名和密码，注册页面输入内容包含用户名、确认密码和密码。整体以蓝白色为主色调，界面简介美观。
```

完成文件创建后，预览文件并查看效果，对于部分细节问题，可以通过DeepSeek&Cline进行修改。

window.onload = function () {
    const loginTab = document.getElementById('loginTab');
    const registerTab = document.getElementById('registerTab');
    const loginContent = document.getElementById('loginContent');
    const registerContent = document.getElementById('registerContent');

    // 添加空值检查
    if (!loginTab || !registerTab || !loginContent || !registerContent) {
        console.error('One or more elements not found');
        return;
    }

    loginTab.addEventListener('click', () => {
        loginTab.classList.add('text-primary', 'border-primary');
        loginTab.classList.remove('text-gray-400', 'border-gray-200');
        registerTab.classList.add('text-gray-400', 'border-gray-200');
        registerTab.classList.remove('text-primary', 'border-primary');
        loginContent.classList.add('active');
        registerContent.classList.remove('active');
    });

    registerTab.addEventListener('click', () => {
        registerTab.classList.add('text-primary', 'border-primary');
        registerTab.classList.remove('text-gray-400', 'border-gray-200');
        loginTab.classList.add('text-gray-400', 'border-gray-200');
        loginTab.classList.remove('text-primary', 'border-primary');
        registerContent.classList.add('active');
        loginContent.classList.remove('active');
    });
}

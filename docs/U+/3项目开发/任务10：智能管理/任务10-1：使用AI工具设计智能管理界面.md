任务描述

**1. 任务描述**

一般的网站通常都有管理员后台，可以修改网站各种数据。传统网站后台的数据维护工作存在效率低、易出错的问题。管理员进行数据操作时，需在多个功能模块间反复切换，手动设置筛选条件、编辑数据，不仅耗费大量时间，还可能因操作繁琐导致数据录入错误。
以糖尿病相关数据管理为例，在传统网站后台，若要统计近一个月咨询糖尿病治疗方案的用户数量，管理员需先进入用户数据查询模块，手动设置时间范围为近一个月，再输入关键词 “糖尿病治疗方案” 进行筛选，最后从导出的表格中逐条统计数据。若要修改糖尿病专科医生的坐诊时间，需在医生信息管理页面，找到对应医生条目，进入编辑模式修改时间，再点击保存，过程十分繁琐。​

而在应用大模型的网站中，管理员只需输入自然语言指令，如 “统计近一个月咨询糖尿病治疗方案的用户数量”，大模型就能快速从数据库中提取并汇总数据；输入 “将王医生的糖尿病门诊时间改为每周二下午”，大模型可自动定位并修改对应数据，无需复杂的手动操作，大幅提升数据维护的效率与准确性。

本任务实现通过AI工具来为管理员设计数据智能管理平台。

任务指导

#### 1. 智能管理界面设计分析

根据任务描述中对智能管理平台的描述，可以分析到智能管理平台包括以下两部分能力，1是将自然语言转换为特定的指令，例如“删除所有无糖尿病的用户风险信息”->系统执行查询和删除操作。2是对网站数据的管理，包括对数据的添加、删除、修改及查询。这样就可以得出结论，在界面中需要有输入框和执行按钮，并且对应增删改功能，需要展示执行结果；对应查询功能则需要通过列表进行展示。除此之外还要考虑执行操作历史记录的查看功能。

function renderObjectTable(data) {
    const container = document.getElementById('dataTableContainer');
    if (!container || !Array.isArray(data) || data.length === 0) return;

    // Create table element
    const table = document.createElement('table');
    table.className = 'w-full min-w-max whitespace-nowrap';

    // Create table header
    const thead = document.createElement('thead');
    const headerRow = document.createElement('tr');
    headerRow.className = 'border-b border-gray-200';

    // Get headers from first object keys
    Object.keys(data[0]).forEach(key => {
        const th = document.createElement('th');
        th.className = 'py-2 px-3 text-left text-sm font-medium text-gray-600';
        th.textContent = key;
        headerRow.appendChild(th);
    });
    thead.appendChild(headerRow);
    table.appendChild(thead);

    // Create table body
    const tbody = document.createElement('tbody');
    data.forEach(item => {
        const row = document.createElement('tr');
        row.className = 'border-b border-gray-100';

        Object.values(item).forEach(value => {
            const td = document.createElement('td');
            td.className = 'py-2 px-3 text-sm text-gray-600';
            td.textContent = typeof value === 'object' ? JSON.stringify(value) : value;
            row.appendChild(td);
        });

        tbody.appendChild(row);
    });
    table.appendChild(tbody);

    // Clear container and append table
    container.innerHTML = '';
    container.appendChild(table);
}

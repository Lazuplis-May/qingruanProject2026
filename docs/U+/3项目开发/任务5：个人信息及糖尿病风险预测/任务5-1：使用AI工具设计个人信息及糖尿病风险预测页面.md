任务指导

#### 1. 功能分析

当在个人中心中点击“个人信息”页面时跳转至个人信息页面，在此页面中展示个人信息，也可以进行信息修改，在修改个人信息时会询问是否患有糖尿病，如果未患有糖尿病，则在填写完个人信息后，还会进行糖尿病风险评估，如下图所示。

![img](https://tecs-prod-static.obs.cn-north-1.myhuaweicloud.com/b6ede7dc667344c9b6cf219f83a9cd69%2Frichtext%2Fimage%2F20250425%2F0e636ae0bcdc4db487d327128771659f.png)

图1 业务流程

个人信息填写内容可参考《中国 2 型糖尿病防治指南（2020 年版）》中中国糖尿病风险评分表，相关内容如下：

**评分指标及分值**

1. **年龄（岁）**：20 - 24 岁为 0 分；25 - 34 岁为 4 分；35 - 39 岁为 8 分；40 - 44 岁为 11 分；45 - 49 岁为 12 分；50 - 54 岁为 13 分；55 - 59 岁为 15 分；60 - 64 岁为 16 分；65 - 74 岁为 18 分。
2. **体质指数（*****kg*****/*****m*****2）**：＜22.0 为 0 分；22.0 - 23.9 为 1 分；24.0 - 29.9 为 3 分；≥30.0 为 5 分。
3. **腰围（cm）**：男＜75.0、女＜70.0 为 0 分；男 75.0 - 79.9、女 70.0 - 74.9 为 3 分；男 80.0 - 84.9、女 75.0 - 79.9 为 5 分；男 85.0 - 89.9、女 80.0 - 84.9 为 7 分；男 90.0 - 94.9、女 85.0 - 89.9 为 8 分；男≥95.0、女≥90.0 为 10 分。
4. **收缩压（mmHg）**：＜110 为 0 分；110 - 119 为 1 分；120 - 129 为 3 分；130 - 139 为 6 分；140 - 149 为 7 分；150 - 159 为 8 分；≥160 为 10 分。
5. **糖尿病家族史（父母、同胞、子女）**：无为 0 分；有为 6 分。
6. **性别**：女为 0 分；男为 2 分。

**适用范围及风险判断**

该评分表适用于对 20 - 74 岁普通人群进行糖尿病风险评估，评分值的范围为 0 - 51 分。总分≥25 分则为高风险糖尿病人群。

以上评分指标包括年龄、身高体重、性别、家族病史、腰围和收缩压，由于腰围和收缩压不太好立即获取，可以通过其他参数来推断，所以腰围和收缩压均为选填项。

由以上分析可知：

个人信息界面包括：年龄、性别、家族病史、身高、体重、腰围、收缩压、是否患病及建议；

个人信息填写界面中包括是否患病、年龄、性别、家族病史、身高、体重、腰围、收缩压；

风险评估结果界面包括糖尿病风险及建议。

个人信息查询功能可通过数据管理工作流来进行实现，个人信息填写保存及风险评估结果可通过Dify工作流编排来实现。

        const typeSection = document.querySelector('.bg-white.rounded-lg.shadow-md.p-4.mt-4:nth-child(2)');

        if (this.value === 'yes') {
            typeSection.style.display = 'block';
        } else {
            typeSection.style.display = 'none';
            selectedType = null;
        }
    });

    // Initialize - hide type selection if default is "no"
    if (diabetesStatus.value === 'no') {
        document.querySelector('.bg-white.rounded-lg.shadow-md.p-4.mt-4:nth-child(2)').style.display = 'none';
    }

    // Handle type selection
    typeButtons.forEach(button => {
        button.addEventListener('click', function() {
            // Remove selection from all buttons
            typeButtons.forEach(btn => {
                btn.classList.remove('border-primary', 'bg-third');
            });
            
            // Add selection to clicked button
            this.classList.add('border-primary', 'bg-third');
            selectedType = this.querySelector('p:first-child').textContent;
        });
    });

    // Handle next button click
    nextBtn.addEventListener('click', function() {
        if (diabetesStatus.value === 'yes' && !selectedType) {
            alert('请选择糖尿病类型');
            return;
        }

        const result = diabetesStatus.value === 'no' ? '否' : selectedType;
        window.location.replace(`informationGathering.html?disease=${encodeURIComponent(result)}`);
    }); 
}

任务指导

##### 1. UI界面设计

UI界面设计是通过视觉元素（如文字、图标、按钮、颜色、布局等）和交互元素（如点击、滑动、滚动等）来构建用户与数字产品之间交互的桥梁。它旨在让用户能够轻松、高效地使用产品，同时提供愉悦的视觉体验。

 

UI界面设计目标如下：

提升用户体验：通过简洁、直观的界面设计，让用户能够快速理解和操作产品。

满足用户需求：根据用户的行为习惯和需求，设计符合用户期望的界面。

增强品牌形象：通过一致的视觉风格和高质量的设计，提升产品的品牌形象和用户信任度。

提高效率：优化界面布局和交互流程，减少用户的学习成本和操作步骤。

适应多平台：确保界面在不同设备（如手机、平板、电脑）和不同屏幕尺寸上都能良好显示。

 

UI界面设计原则如下：

简洁性：界面设计应尽量简洁，避免过多的视觉元素和复杂的交互逻辑，让用户能够快速找到所需功能。

一致性：保持界面元素的风格、布局和交互方式一致，避免让用户感到困惑。

直观性：界面应直观易懂，用户无需过多学习即可使用。例如，使用常见的图标和按钮布局。

反馈性：用户操作后应有明确的反馈，如按钮点击后的视觉变化、加载动画等。

可访问性：设计应考虑不同用户群体的需求，包括残障人士，确保界面易于访问和操作。

灵活性：界面设计应适应不同用户的需求和操作习惯，提供多种操作方式。

美观性：界面应具有良好的视觉效果，包括色彩搭配、排版和图形设计。

在js目录中创建index.js，实现页面切换功能，代码如下：

```
window.onload = function () {
    // 导航栏点击处理
    document.querySelectorAll('.tab-bar a').forEach(link => {
        link.addEventListener('click', (e) => {
            e.preventDefault();

            // 移除所有active样式
            document.querySelectorAll('.tab-bar a').forEach(el => {
                el.classList.remove('text-primary', 'active');
                el.classList.add('text-gray-500');
            });

            // 添加当前active样式
            link.classList.add('text-primary', 'active');
            link.classList.remove('text-gray-500');

            // 加载对应页面
            const targetPage = link.getAttribute('data-target');
            document.querySelector('iframe').src = targetPage;
        });
    });
}
```

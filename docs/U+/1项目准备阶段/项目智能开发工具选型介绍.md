任务描述

**1. 任务描述**

近年来大语言模型(LLMs)技术及工具如雨后春笋得到快速发展，如ChatGPT、DeepSeek、Copilot、Cursor等等。它们对软件工程领域产生了极大冲击，不仅可为软件开发和运维的各项活动提供智能化支持，提升软件开发的效率和质量，而且改变软件开发方式，产生新的软件开发范型。

随着 LLMs 大模型技术的迅猛发展，软件开发领域迎来了重大变革。如今，开发者不仅能依靠传统代码编写来实现软件功能，借助大模型及其流程编排，更能赋予软件超乎以往的强大能力。Dify 作为一款领先的 LLM 应用开发平台，集成了 DeepSeek、ChatGPT、豆包等丰富多样的前沿模型，开发者通过灵活调用这些模型，并搭配平台提供的可视化 Prompt 编排、插件扩展等功能，即可轻松搭建各类复杂应用，实现智能对话、内容创作、数据分析等多元且强大的功能，极大提升开发效率，拓展软件应用边界 。

在IT业界，使用大模型工具来辅助软件开发已经成为重要的趋势，结合大模型工具来开发软件已成为每个软件工程师必备的能力。

本任务结合项目需求，将分析出适合本项目使用的智能开发工具，在后续的开发中进行使用。

任务指导

**1. 大模型发展现状**

大模型近几年经历了从2017年Transformer架构的提出到2022年ChatGPT模型的推出，实现了从技术奠基到广泛应用的快速演进。2023年以来，大模型技术快速发展，多模态模型和应用场景不断拓展，国内外均涌现出众多杰出的大模型。

2023年OpenAI的GPT-4模型脱颖而出，成为无可争议的最佳AI大模型之一。GPT-4以其卓越的自然语言处理能力、复杂的推理能力、高级编码能力以及多元学术学习能力，赢得了业界的广泛赞誉。同时，它也是首个能同时处理文本和图像的多模态模型，这一创新功能使得它在众多AI模型中独树一帜。此外，谷歌的PaLM 2模型也备受瞩目，它在常识推理、形式逻辑、数学能力等方面表现出色，并支持20余种语言的高级编码。

而在国内，大模型同样呈现出爆发式增长。特别是在2025年，中国推出的DeepSeek-R1大型语言模型，以其超成本效益和开源设计，推动了先进大模型的普及。DeepSeek-R1采用大规模专家混合（MoE）架构，参数规模高达6710亿，性能比肩GPT-o1，展现了非凡的实力。此外，华为、阿里等企业也发布了各自的大语言模型，进一步丰富了国内大模型的生态。

国内也纷纷推出了大模型工具，如DeepSeek、豆包的MarsCode、阿里的通义灵码等等。

DeepSeek开发的大模型基于Transformer架构，采用混合专家模型等技术，推理能力强，每秒可生成60个tokens，在数学推理、代码生成等领域表现出色，并且完全开源，支持本地部署，用户可定制模型并掌控数据隐私，API服务价格亲民，成为软件开发时炙手可热的大模型工具，基于以上特点，本次项目我们使用DeepSeek大模型进行辅助开发。

**2. DeepSeek融入VSCode**

DeepSeek融入VSCode需要用到编程插件，本次将使用Cline插件进行前端页面辅助开发。

作为一款使用方便，功能强大的智能编程助手，Cline为开发者提供了一系列强大的功能：

- 智能代码分析与生成：Cline能够分析项目的文件结构和源代码抽象语法树（AST），通过正则表达式搜索和读取相关文件，快速了解现有项目，能够处理复杂的软件开发任务，逐步完成文件操作与错误处理；
- 可以创建和编辑文件；
- 实时监控linter/编译器错误；
- 能够主动修复诸如缺少导入和语法错误等问题；
- 终端命令执行：直接在用户终端中执行命令并监控输出；
- 能够对开发服务器问题等进行反应和处理；
- 支持多种API提供商，如DeepSeek、OpenRouter、Anthropic、OpenAI、Google Gemini等，可配置任何兼容OpenAI的API，支持通过Ollama使用本地模型；
- 成本追踪：跟踪整个任务循环和单个请求的总token数和API使用成本，让用户随时了解开支情况。

 

Cline的这些功能使其成为一个全面的编程助手，能够在项目开发的各个阶段为开发者提供支持。

**3.LLM应用开发平台**

LLM 是大型语言模型（Large Language Model）的简称，而LLM应用开发平台则是基于大模型进行应用开发，应用包括工作流、智能体、知识库等，并提供API的方式以供前端页面进行调用。

市面上的LLM应用开发平台包括“Coze”、“Dify”、“FastGPT”等。

| 平台名称 | 核心功能                                                     |
| -------- | ------------------------------------------------------------ |
| Dify     | 1. 可视化编排 AI 工作流，如 Chatflow 适用于对话类情景。 2. 支持数百种 LLMs，涵盖 GPT、Mistral、Llama3 等及 OpenAI API 兼容模型。 3. 具备 RAG 功能，可从常见文档格式提取文本。 4. 提供 50 + 内置工具给 AI Agent，如谷歌搜索等。 |
| FastGPT  | 1. 专注知识库训练，支持创建 RAG 系统，提供自动化工作流程编排。 2. 对外 API 接口对齐 OpenAI 官方接口，可接入现有 GPT 应用，也能集成到企业微信、公众号、飞书等平台。 |
| Coze     | 1. 多语言模型支持，使用如 GPT-4-8K、GPT-4-128K 和云雀语言模型等2。 2. 插件系统丰富，集成超 60 款插件，涵盖资讯阅读、旅游出行等功能，支持自定义插件2。 3. 具备知识库功能，可上传多种格式文档，或基于 URL 获取在线内容和 API JSON 数据2。 4. 有数据库和记忆能力，允许 Bot 访问会话内存和上下文 |

由于Dify自定义程度高，开源、模型兼容性强的特点，在本项目中通过Dify平台进行服务端开发。

**4. AI辅助页面设计**

UI界面设计AI工具目前主流的有MasterGo AI 、墨刀 AI、Figma AI、V0.dev等。

| 工具名称    | 特点                                                         |
| ----------- | ------------------------------------------------------------ |
| MasterGo AI | 集成多种设计软件功能，涵盖界面设计、交互原型及设计系统三大核心功能，具备图形绘制、自动布局、交互演示、智能动画等工具，还有设计工程化功能可将设计稿秒转代码，AI 辅助设计功能包括设计规范提取、设计检查和图片增强等。 |
| 墨刀 AI     | 集成 AI 技术的原型设计工具，提供从需求文档到原型线框、流程梳理等全流程支持，功能涵盖 AI 对话、生成组件、生成原型页面、文字智能填充等，特色功能有智能抠图、配色、自然语言操作等，支持多种平台。 |
| Figma AI    | 强大的设计辅助工具，功能包括快速搜索设计资产、自动重命名和整理图层、生成和替换文本内容、移除图像背景、重写和翻译文案、生成分类和总结等，紧密围绕设计流程，智能理解设计意图。 |
| V0.dev      | Vercel 推出的以 AI 模型生成交互页面的工具，支持 React、Vue、Svelte 等前端框架，可使用 Tailwind CSS 进行布局，能通过简单提示词生成多样用户界面，并可通过小组件微调代码，还具备实时协作、设计共享等功能。 |

根据使用难易度、免费额度、效率等因素，本次使用MasterGo AI来完成页面设计任务。MasterGo AI可以根据提示词设计出多版UI界面供用户选择，并可在此基础上继续向其发出提示词进行页面修改优化。不仅如此，完成的UI页面可以自动生成HTML或VUE代码，极大程度上提高了开发效率。

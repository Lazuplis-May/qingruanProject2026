任务指导

#### 1. 下载Nginx

本项目通过虚拟机或云沙箱的方式提供服务端，当前端界面编写完毕后，也需要通过虚拟机或云沙箱的方式完成部署，在部署前首先要安装部署软件，本项目使用Nginx软件进行项目部署，进入到虚拟机或云沙箱界面，并打开命令行，执行以下命令下载nginx。

```
yum install -y nginx
```

执行效果如下图所示：

![img](https://tecs-prod-static.obs.cn-north-1.myhuaweicloud.com/b6ede7dc667344c9b6cf219f83a9cd69%2Frichtext%2Fimage%2F20250423%2Fc91681685d16441d90050964a3263c51.png)

图1 安装Nginx

由于nginx的默认端口与Dify的端口冲突，还需要修改nginx的端口配置，执行以下命令打开配置文件。

```
vim /etc/nginx/nginx.conf
```

执行结果如下图所示：

![img](https://tecs-prod-static.obs.cn-north-1.myhuaweicloud.com/b6ede7dc667344c9b6cf219f83a9cd69%2Frichtext%2Fimage%2F20250423%2F833f3e9c967643e2a09450517aa79582.png)

图2 配置Nginx

修改server->listen中的80，将值修改为8080，按下i进入编辑状态，使用上下左右键移动输入光标，修改完成后按下“esc”键，然后输入“:wq”并按下回车键保存。修改后如下图所示。

![img](https://tecs-prod-static.obs.cn-north-1.myhuaweicloud.com/b6ede7dc667344c9b6cf219f83a9cd69%2Frichtext%2Fimage%2F20250423%2F02d79654571247fbb78739715b659cdd.png)

图3 配置Nginx

执行以下命令启动nginx，并设置为开机自启动。

```
systemctl start nginx
systemctl enable nginx
```

接下来验证启动是否成功，在本地打开浏览器，并在地址栏中输入“虚拟机ip:8080”（云沙箱环境则输入8080端口映射地址），如下图所示则为启动成功。

![img](https://tecs-prod-static.obs.cn-north-1.myhuaweicloud.com/b6ede7dc667344c9b6cf219f83a9cd69%2Frichtext%2Fimage%2F20250423%2F6ae10e84e43444cdbd164780055e500c.png)

图4 测试Nginx

#### 2. 部署Web项目

在本项目中，通过git拉取代码并部署，在部署前将所有代码推送到CodeArts中。

首先在虚拟机或云沙箱中安装git，命令如下。

```
yum install -y git
```

然后将项目克隆到虚拟机或沙箱目录中，命令如下。

```
git clone xxx
```

其中xxx表示CodeArts仓库地址，可在如下图所示位置进行复制。

![img](https://tecs-prod-static.obs.cn-north-1.myhuaweicloud.com/b6ede7dc667344c9b6cf219f83a9cd69%2Frichtext%2Fimage%2F20250423%2F65a04aeae9d549f7b4865838c81916d2.png)

图5 获取git地址

执行命令时会提示输入用户名和密码，输入CodeArts代码托管的用户名和密码即可，详情可参考任务1-3中的项目关联认证部分。

克隆完成后，如下图所示。

![img](https://tecs-prod-static.obs.cn-north-1.myhuaweicloud.com/b6ede7dc667344c9b6cf219f83a9cd69%2Frichtext%2Fimage%2F20250423%2F58a3491173cc4a329a03ac463a2ad887.png)

接下来将项目代码部署到nginx中，nginx默认部署位置为“/usr/share/nginx/html/”，执行以下命令删除原先的nginx中配置的HTML文件。

```
rm -rf  /usr/share/nginx/html/*
```

然后执行以下命令将代码拷贝到nginx部署目录中，命令如下（diabetesAssistant为CodeArts代码仓库的仓库名）。

```
cp -r diabetesAssistant/* /usr/share/nginx/html/
```

拷贝完成后，刷新刚刚访问的8080端口，最终效果如下。
